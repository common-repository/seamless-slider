In case you will need to manipulate slider outside of it's context. You can use simple API:

  • _SS.getSlider( slider_alias ) - returns current slider instance by alias. In case there are more than one instances of the slider, returns array.
  • _SS.getSlider( slider_alias ).nextSlide() - proceeds to next slide.
  • _SS.getSlider( slider_alias ).previousSlide() - reverts to previous slide.
  • _SS.getSlider( slider_alias ).setSlide( slideNumber ) - sets active slide to provided slideNumber.
