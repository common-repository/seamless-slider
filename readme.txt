== Description ==

Seamless Slider is created with the goal to make simple slider creation fast and comfortable,with no requirement of purchasing premium sliders.

Delivering highest quality, Seamless Slider is the right thing for you if you need to set up a simple slider fast.

**Main Features**

* Designed with Wordpress styles
* Use social media for images ( Instagram )
* Fully AJAX based - delivering fast usability
* Optimized for SEO
* Responsive Design
* Cross Browser Compatibility
* Different transitions
* No security exploits
* Rich customization
* Navigation
* Slide sorting
* Compatibility with any theme
* Compatibility with Multisite
* Doesn't conflict with other plugins

Here's a link to [Demo](http://guramidev.com/plugins/wordpress/ "Seamless Slider")

== Installation ==

1. Go the the Wordpress Admin Panel.
2. On the left side menu select Plugins > Add New
3. In "Search Plugins" field enter "Seamless Slider" and search
4. Press "Install Now" button of the "Seamless Slider"
